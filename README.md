Intelligent Voice
=======

Intelligent Voice has been renamed to VFC Communications Analytics

For more information on VFC Communications Analytics see:
https://www.verint.com/compliance-solutions/verint-for-financial-compliance/communications-analytics/

Support
=======

The support portal previously at https://support.intelligentvoice.com has been retired.

To get support for VFC Communications Analytics please first login to https://connect.verint.com and then:

1. To create a new support case go to https://connect.verint.com/p/createcase?productShortCode=vfc
2. To view documentation, release notes and download links go to https://connect.verint.com/support/vfc/w/coman/
3. For knowlegebase articles go to https://connect.verint.com/support/vfc and filter by the "Speech Transcription" component
